## 📌 About The Project

AI Mitra is a web-based personal AI assistant that allows users to interact
with multiple AI models through text or voice input. Built as a full-stack
web application, it provides a clean, responsive interface similar to
ChatGPT or Gemini — built from scratch by college students.

---
## ✨ Features

- 🎙️ **Voice Input** — Speak your question, it converts to text automatically
- 🤖 **Multiple AI Models** — Gemini 2.5 Flash, Gemini 2.0 Flash, Gemini 2.5 Pro, Cohere Command R
- 💬 **Chat History** — Recent conversations saved when logged in
- 🔐 **Google Login** — One click sign in with Google
- 🎨 **3 Themes** — Light, Dark and more — personalize your experience
- 📧 **Forgot Password** — Email verification code for secure password reset
- 📝 **Feedback System** — Users can submit feedback directly
- ❓ **Help & Support Page** — Dedicated support page for users
- ℹ️ **About Page** — Learn more about the platform
- 📱 **Fully Responsive** — Works perfectly on mobile and desktop

## 🚀 Getting Started

### Prerequisites
- Node.js installed
- MongoDB account
- Gemini API key (free from Google AI Studio)
- Cohere API key (free from cohere.com)

### Installation

1. **Clone the repository**
```bash
git clone https://github.com/YOURUSERNAME/AIMitra.git
cd AIMitra
```
2. **Install dependencies**
```bash
npm install
```

3. **Create a `.env` file in root directory**
```env
MONGODB_URI=your_mongodb_connection_string
GEMINI_API_KEY=your_gemini_api_key
COHERE_API_KEY=your_cohere_api_key
GOOGLE_CLIENT_ID=your_google_client_id
GOOGLE_CLIENT_SECRET=your_google_client_secret
JWT_SECRET=your_jwt_secret
```
4. **Run the application**
```bash
npm start
```

5. Open your browser and go to `http://localhost:3000`
