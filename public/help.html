Help & Support

Everything you need to know about using AI Mitra

Features & Capabilities

  • Chat Interface: Type prompts directly into the message bar to converse with AI Mitra. The layout centers on a landing page and smoothly transitions into a standard chat stream once you begin.
  • Smart Suggestions: Fast-track your queries using the pre-written chip buttons (e.g., "Write a blog post", "Explain quantum computing") when starting a new session.
  • New Chat: Instantly clear the canvas and start a pristine conversational thread by clicking the "New Chat" button in the sidebar.
  • Recent Chats: Access a history of your previous sessions within the collapsible left navigation pane.
  • Settings Menu: A centralized dropdown at the bottom-left gear icon grants access to your Profile, Feedback panel, and Logout options.
  • Feedback System: An intuitive, dimmed modal box allowing you to submit application feedback. It intelligently ensures you've actually typed something before enabling submission.
  • Theme Support: Effortlessly toggle between meticulously crafted Light and Dark themes via the sidebar icon.
  • Responsive Interaction: Optimized for all devices, whether you're using voice transcription on mobile or a wide desktop monitor.
